# Explainable AI in Semiconductor Manufacturing

This repository contains a production-ready Python implementation of the two-step framework proposed in the research paper *Using Explainable Artificial Intelligence to Improve Process Quality* (Senoner et al., 2021). Designed to handle the high-dimensional, nonlinear data typical of semiconductor fabrication, the pipeline uses a Gradient Boosting metamodel (XGBoost) combined with SHAP (Shapley Additive Explanations) to open the ML "black box." It automatically identifies which specific manufacturing processes are causing the most yield variance and then prescribes targeted, physical improvement actions—such as altering machine routing—to minimize defects and boost overall production yield.

Built with MLOps best practices, the codebase is fully modularized into separate layers for data generation, defensive ingestion, model training, and SHAP-based explainability. To test the framework locally (requires `pandas`, `numpy`, `xgboost`, and `shap`), first run `python generate_data.py` to simulate a factory floor with a hidden, nonlinear machine defect. Then, execute `python main.py` to watch the AI cut through the noise of 50 continuous parameters, prioritize the failing process, and mathematically prove which machine needs to be avoided.
